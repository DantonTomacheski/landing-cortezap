No Brasil, "sistema para barbearia com WhatsApp" virou critério de compra. O problema é que "com WhatsApp" significa coisas muito diferentes dependendo do sistema. Desde notificação por WhatsApp até bot conversacional com API Oficial da Meta. A diferença entre esses extremos é a diferença entre um lembrete e uma recepcionista virtual 24 horas.

Vamos separar o que funciona do que é marketing.

## Os 3 níveis de "WhatsApp" em sistemas de barbearia

### Nível 1: Notificações por WhatsApp
O sistema envia mensagens automáticas para o cliente: confirmação de agendamento, lembrete de horário, aviso de cancelamento. O cliente recebe, mas não pode interagir de volta pelo mesmo canal para agendar.

**Quem faz:** Trinks, Barberify, AgendaJá e outros. Boa parte do mercado.

**O que resolve:** Reduz no-show com lembretes. É útil, mas não automatiza o atendimento.

**O que não resolve:** O cliente que manda "tem horário amanhã?" ainda precisa de resposta humana.

### Nível 2: Bot não oficial
O sistema usa APIs não autorizadas pela Meta para rodar um bot no WhatsApp. O bot responde, agenda, interage. Funciona — até o número ser banido.

**Quem faz:** Vários sistemas menores e soluções independentes. Nem sempre divulgam que a API é não oficial.

**O que resolve:** Automatiza o atendimento. O bot faz o trabalho de uma recepcionista.

**O risco:** Banimento do número. A Meta identifica uso não oficial e pode bloquear permanentemente. Para uma barbearia que construiu base de clientes naquele número, é perda grave.

### Nível 3: Bot com API Oficial da Meta
O sistema usa a WhatsApp Business API oficial, aprovada pela Meta. Bot conversacional que agenda, reagenda, cancela. Sem risco de banimento porque está dentro das regras.

**Quem faz:** CorteZap é um dos poucos sistemas de barbearia no Brasil que usa API Oficial para bot conversacional.

**O que resolve:** Automatiza o atendimento com segurança. O número está protegido. O bot funciona 24 horas.

**O custo:** API Oficial tem custo por mensagem (cobrado pela Meta). No CorteZap, esse custo está embutido nos planos.

## O que importa na prática

**Se você já perdeu número:** Sabe o custo. Reconstruir a lista de contatos, avisar clientes um a um pelo novo número, perder o histórico de conversas. API Oficial elimina esse risco.

**Se nunca perdeu número:** Pode parecer precaução desnecessária. Até acontecer. A Meta não avisa — simplesmente bloqueia. E o recurso raramente resulta em desbloqueio para APIs não oficiais.

**Volume de atendimento importa:** Se você recebe 5 mensagens por dia pedindo horário, responder manualmente é viável. Se recebe 30, o bot economiza 2-3 horas diárias. Multiplique por 22 dias úteis.

**Horário de atendimento:** O bot atende de madrugada, domingo, feriado. O cliente que lembrou de marcar à meia-noite agenda na hora. Sem bot, essa mensagem espera até você ver — e o cliente pode ter marcado em outro lugar.

## Perguntas para avaliar um bot de WhatsApp

Antes de contratar qualquer sistema com "bot de WhatsApp", pergunte:

1. **Qual API vocês usam?** API Oficial da Meta ou outra solução? Se não responderem claramente, desconfie.
2. **O bot agenda ou só notifica?** Notificar é diferente de conversar e agendar.
3. **O que acontece se o número for banido?** Se usam API oficial, esse cenário não existe. Se não usam, qual é o plano B?
4. **O custo de mensagens está incluso?** API Oficial tem custo por mensagem. Está no plano ou é cobrado à parte?
5. **O bot entende linguagem natural ou só menu?** Bot de menu (1 para agendar, 2 para cancelar) funciona, mas bot que entende "quero cortar amanhã às 15h" é mais fluido.
